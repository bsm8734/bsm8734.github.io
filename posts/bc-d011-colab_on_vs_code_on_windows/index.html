VS code에서 colab 실행

Colab

  • 다음의 코드를 실행 !pip install colab-ssh --upgrade from colab_ssh import launch_ssh_cloudflared, init_git_cloudflared launch_ssh_cloudflared(password="패스워드")
  • VSCode Remote SSH 에서 url 복사

VS Code

  • ctrl + shift + P
  • Remote-SSH: Connect to Host... 선택
  • url 넣기 ➡ Linux ➡ continue ➡ 비밀번호 입력
  • open folder (/content/drive/MyDrive/Colab Notebooks/)
