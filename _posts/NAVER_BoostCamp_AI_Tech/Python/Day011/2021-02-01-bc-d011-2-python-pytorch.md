##### <span style="color:darkblue">Pytorch = Numpy + AutoGrad + Function</span>  

- Numpy: Numpy 구조를 가지는 Tensor 객체로 array 표현
- Auto Gradient: 자동미분을 지원하여 DL 연산을 지원
- Function: 다양한 형태의 DL을 지원하는 함수와 모델을 지원
